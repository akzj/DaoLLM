# 更新日志

## v0.1.0 (初始版本)

### 核心功能

- ✅ **多Agent架构**
  - SystemAgent：资源管理、场景生成、选项生成
  - DirectorAgent：发言顺序决策、玩家响应决策
  - RoleAgent：角色发言异步生成

- ✅ **记忆系统**
  - 混合压缩策略（重要性排序+语义相似度去重+时间衰减）
  - Token控制（≤1024）
  - 记忆检索和上下文管理

- ✅ **资源管理**
  - 4bit量化支持
  - 动态LoRA加载/卸载
  - LoRA锁机制（线程安全）

- ✅ **配置系统**
  - YAML格式配置（剧本+角色）
  - 配置验证工具
  - 配置加载工具

- ✅ **玩家交互**
  - 异步玩家输入处理
  - 选项生成和选择
  - 超时和取消支持

- ✅ **工具和测试**
  - 性能监控工具
  - 日志工具
  - 错误处理工具
  - 简单测试脚本

### 技术特性

- Qwen模型格式支持（ChatML格式）
- 异步生成（asyncio + 线程池）
- 配置验证
- 错误处理机制

### 已知限制

- 玩家输入目前使用命令行，可扩展为GUI/Web
- 记忆重要性计算使用规则，可优化为模型评分
- 导演Agent决策可进一步优化Prompt
