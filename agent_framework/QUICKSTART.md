# 快速启动指南

## 运行方式

### 方式1：使用 Python 运行（推荐）

```bash
cd agent_framework
python run.py
```

### 方式2：直接执行脚本

```bash
cd agent_framework
chmod +x run.py  # 首次运行需要添加执行权限
./run.py
```

## 玩家输入说明

当出现玩家选项时：

1. **输入选项编号**：直接输入数字（如 `1`、`2`、`3`）
2. **输入选项内容**：输入选项的完整文本或部分文本
3. **避免输入命令**：不要输入 shell 命令（如 `./run.py`、`python` 等）

### 示例

```
[玩家选项]
1. 询问秘境情况
2. 询问队伍实力
3. 保持沉默
请输入选项编号（1-3）或直接输入选项内容：
> 1          # ✓ 正确：输入编号
> 询问秘境   # ✓ 正确：输入部分文本
> ./run.py   # ✗ 错误：这是命令，会被拒绝
```

## 常见问题

### Q: 看到 "LoRA未找到" 的提示？

A: 这是正常的。如果还没有训练LoRA权重，系统会自动使用基础模型。不影响游戏运行。

### Q: 输入被错误识别？

A: 如果输入了命令或无效内容，系统会提示重试。最多可以重试3次。

### Q: 如何中断游戏？

A: 按 `Ctrl+C` 可以中断程序。
